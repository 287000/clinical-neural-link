🛠 Features

Admin Hub: A hidden control panel for instructors to deploy new assessments.

Live Presence: Real-time monitoring of active users in the portal.

JSON Deployment: Rapid quiz creation using structured JSON data.

Responsive Design: Optimized for mobile, tablet, and desktop clinical environments.

🔐 Admin Access

To access the Admin Hub (System Master Control):

Navigate to the Login screen.

Enter your name (e.g., "Danny Phiri").

Enter the Master Key: ADMIN2026 in the Access Key field.

Click Establish Connection.

Once logged in, click the Admin Hub button in the top navigation bar.

📄 Quiz JSON Schema

To deploy a quiz in the Admin Hub, use the following format:

{
  "questions": [
    {
      "q": "What is the primary site of the TCA cycle?",
      "options": ["Cytosol", "Mitochondrial Matrix", "Ribosome", "Nucleus"],
      "answer": 1,
      "reason": "The TCA cycle occurs in the mitochondrial matrix where the necessary enzymes are located."
    }
  ]
}
